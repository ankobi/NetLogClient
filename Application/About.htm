Version 2.0 beta 6 (Sept 20, 2011).  Check for the latest version on our project page.

About

An open source UDP log event viewer written in C#.  This application should be suitable for use with Apache's log4j, log4net, NLog and other similar logging frameworks that sends UDP log messages conforming to the log4j schema.
